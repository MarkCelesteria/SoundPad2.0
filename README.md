# SoundPad 2.0

> Note: This project is intended for school use (e.g., roleplaying scenarios) where fast, responsive soundpads and saved background music sounds are needed.

## Description
A lightweight soundpad web app for quickly playing and managing audio clips.

## Usage
- Add your sound files in the `assets/` folder.
- Configure buttons in `index.html` and `js/script.js`.
- Use this in classroom/roleplay setups to switch background audio and sound effects fast.
